wp-svg-upload

Adds full SVG media support to WordPress.

Download as .zip Download as .tar.gz View on GitHub

Upload SVGs to WordPress Media Library

This plugin will enable your WordPress media library to support SVG files. Install using composer or add it to your plugins directory. Once the plugin is enabled, you're good to go! WordPress will support SVGs.
